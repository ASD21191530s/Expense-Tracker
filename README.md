# Expense Tracker

A web-based Expense Tracker built using PHP and MySQL.

## Features
- User Login Authentication
- Add Transactions
- Expense History
- Dashboard
- Session Handling
- Logout Protection

## Tech Stack
- PHP
- MySQL
- HTML
- CSS
- JavaScript
- Bootstrap
